Ajouter une barre de recherche a votre siteweb

Copiez et collez le code ci-dessous dans les pages de votre site Web pour afficher une boîte de recherche Resulti. 

   
   
     <form method="get" action="https://fr.resulti.net/search.html">
	<div>
		<a href="https://resulti.net"> 
			<img src="https://resulti.net/Untitled.png" style="height:100px;padding-bottom:1px;">
		</a>
	</div>
	<input size="30" name="q" placeholder="Rechercher le web avec Resulti...">  
	<input type="submit" value="Search"> 
</form>
